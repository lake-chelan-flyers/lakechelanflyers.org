Today we're moving off of WordPress onto a new website built with Jekyll, a
static website generator. This has several advantages:
* Free hosting on GitHub Pages.
* Faster load times.
* Simpler maintenance.
* A clear history of all changes and the ability to easily roll back.
* The ability to easily preview changes before deploying them.
* Significantly lower chance that the website is compromised in some way.

The contents of the old website are
[available](https://web.archive.org/web/20260422153946/https://lakechelanflyers.org/)
on the WayBack Machine.

The source for this new website is
[available](https://github.com/lake-chelan-flyers/lakechelanflyers.org) on
GitHub.
